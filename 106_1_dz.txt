Перед выполнением заданий важно самостоятельно повторить шаги, выполняемые на уроке.

1) Рассмотреть файл конфиуграции X в текущем исполнении.
# /usr/share/X11/xorg.conf.d в графической.
yum install xorg
sudo X –configure
cat ~/xorg.conf.new
2) Включить и отключить графическую систему на ПК.
startx
firefox
exit
3) Запустить пять процессов в разных консолях.
startx
Ctrl+Alt+T                                                        # Для старта графического терминала 
startx -- :2                                                      # Для запуска графики на указанном терминале
startx -- :3
startx -- :4
startx -- :5
Ctrl+Alt+F1, Ctrl+Alt+F2 и т.д для смены терминала
В каждом терминале запускаем любую команду, например firefox
4) Настроить вывод графики с одного ПК на другой по сети.
xhost +192.168.0.1:0.0
