Перед выполнением заданий важно самостоятельно повторить шаги, выполняемые на уроке.

1) Включить общий доступ к любому из принтеров и подключить его к другому Linux по сети;
apt install cups-pdf
cd /etc/cups
nano printers.conf  # Настройки каждого принтера
# В строке Listen localhost:631 меняем на 0.0.0.0:631 чтобы принимал соединение по всем ip
# Строку Browsing off смени на on чтобы в локалке видеть ппинтеры
# Browsing off на on смени

nano cupsd.conf  # Конфигурация сервера и настроек доступа к нему 
DeviceURI sosket://192.168.0.99:9100    # Веб доступп для принтера и как к нему подключаемся

2) Послать задания на печать по сети с другого Linux и посмотреть где появится очередь печати (на машине где задание 
было послано, или на машине, с которой расшарен принтер);
lpr -PИмя_ПРинтера файл
# Посмотрим очередь печати
lpq -a 
# Посмотрим номер задания на печать
ls /var/spool/cups

3) Подключиться к принтеру, расшаренному в первом шаге с машины Windows и попробовать послать что-нибудь на печать;
# Подключаемся через веб-интерфейс и печатаем

4) При помощи консоли изменить принтер по умолчанию.
nano /etc/cups/lpoptions
# Пропиши в Default <имя принтера>

5) Включить двустороннюю печать на каком-нибудь из созданных принтеров.
lpr -P ljduplex -o sides=two-sided-short-edge file_name

6) Изменить размер полей на каком-нибудь из установленных принтеров. 
Изменить параметр ImageableArea A4/A4: "12 12 583 830" Например на *ImageableArea A4/A4: "12 12 595 842"
