Перед выполнением заданий важно самостоятельно повторить шаги, выполняемые на уроке.

1) Попробовать команды работы с очередью писем: посмотреть письма в очереди, остановить очередь, перезапустить ее.
apt install postfix mailutils


2) Включить переадрессацию писем для root своему пользователю.
nano /etc/aliases
# Впиши:
root: user
# Введи в консоль:
sudo newaliases

3) Ответить на вхоядщее письмо из командной строки.
mail user
# Cc - кому отправить копию
# Subject -тема письма
# Текст письма
# Сtrl^D
mail
# или mail -s «Тест» privet@example.com

4) Удалить входящее письмо, после чего восстановить его.
mail
# Выбери номер письма
d
q
mail
# Выбери номер письма
Скопируй файл электронной почты (т.е.. 141443305.abc123abc.imap01.xyz.com: 2,S) из /home/<user>/Maildir/cur в /home/<user>/Maildir/new и переименуйте его, удалив :2,D суффикс. Этот процесс помечает прочитанные письма как удалённые


5) Прочеть входящее писмьо, после чего отметить его как непрочитанное.
Скопируй файл электронной почты (т.е.. 141443305.abc123abc.imap01.xyz.com: 2,S) из /home/<user>/Maildir/cur в /home/<user>/Maildir/new и переименуйте его, удалив :2,S суффикс. Этот процесс помечает прочитанные письма как непрочитанные


6) Попробовать установить любой другой MTA, кроме postfix.
apt install sendmail
