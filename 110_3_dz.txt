Перед выполнением заданий важно самостоятельно повторить шаги, выполняемые на уроке.

1) Скачать открытый ключ удаленного сервера и добавить его в доверенные.
ssh user@ip
# Согласись с fingerprint 'yes'

2) Попробовать заменить удаленный сервер (указать его ip другому серверу) и попробовать подключиться, используя ключ старого сервера.
Если кто-то влезет в нашу сеть, подменив ip-адрес, при подключении к новому устройству с тами же ip появится запрос на создание fingerprint, cпрашивая, доверять или нет.

3) Создать открытый ключ для всей системы, указать его как доверенный удаленному серверу и попробовать подключаться с локальной машины от имени разных пользователей.
cat ./.ssh/known_hosts >> /etc/.ssh/known_hosts

4) Запустить на локальной машине окно настроек системы удаленного сервера (если удаленный сервер поддерживает X11).
Для запуска графических приложений с удаленного сервера используется ключ –X, например: 
ssh –X 10.0.1.6	     # (запустить сеанс ssh с удаленным сервером 10.0.1.6 перенаправляя X11);
firefox		           #   (запустить на локальной машине окно firefox с удаленного сервера).


5) Пробросить на локальный порт 4567 удаленный порт 443 выбранного веб-сервера (например google) и проверить его работу.
ssh 10.0.1.6 –N –L 4567:213.180.204.3:443 

6) Зашифровать на локальной машине несколько файлов и отправить их на расшифровку удаленной системе.
gpg --gen-key                              # Cоздадим ключи
gpg --export publickey > gpg.public        # Cохраним ключ в файл
scp gpg.public user@192.68.1.3:/home/user  # Скачаем ключ на удлённый хост
gpg --import gpg.public                    # На удалённой машине экспортируем ключ
gpg --list-key                             # На удалённой машине  проверем экспортиованый ключ
gpg --out secret.file --recipient 'publickey' -encrypt file1  # зашивровать file1, сделав выходной файл secret.file, искать по названию ключа 'publickey'
scp user@192.68.1.3:/home/secret.file  /home                  # Скачай с удалённого серера в домашнюю папку зашифрованный файл
gpg --out some.file --decrypt secret.file              # после того,как мы с удалённого хоста скачали зашифрованный файл: scp iphone@192.168.1.5:/home/iphone/gpg.public /home  ,расштфруем его в файл some.file
