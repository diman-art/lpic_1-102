Перед выполнением заданий важно самостоятельно повторить шаги, выполняемые на уроке.

1) Проверить работу всех указанных в уроке возможностей клавиатуры.
Можно включить "Onscreen keyboard" для вывода клавиатуры на экран, "Screen reader" для озвучивания информации на экране, "High contrast" - повышенный констаст для слабовидящих.
All Setting -> Universal Access -> Hight contrast(Высокий контраст) , Large Text(Увеличенный шрифт) , Beep on caps (гудок при нажатии Caps Look/Num Lock)
All Setting -> Universal Access -> Hearing -> Visual Allerts(Подаёт сигнал в случае ошибки -для слабослышащих)
All Setting -> Universal Access -> Typing -> Sticky keys (Можно убрать залипающие клавиши, нажимать их можно поочерёдно)
All Setting - >Universal Access -> Typing - > Slow keys ((Если нажали и удерживаем клавишу система прочитает с регуляцией как долго удерживать клавишу)
All Setting -> Universal Access -> Typing -> Bounce Keys (От случайных быстрых нажатий клавишь-будут игнорироваться с регулированием интервала нажатия клавишь)
All Setting -> Universal Access -> Typing -> Enable by keyboard (Включать функциолнал при помощи клавиатуры)

2) Проверить работу всех указанных в уроке возможностей мыши.
All Setting -> Universal Access -> Pointing and klicking -> Mouse keys (Управлять курсором с помощью num Lock)
All Setting -> Universal Access -> Pointing and klicking -> Simulation Sexondary click (Симулируем двойной клик если удерживаем нажатую кнопку мыши левую)
All Setting -> Universal Access -> Pointing and klicking -> Hower click (Если мы задерживаем курсор мыши на определённый период времени Выполняется дабл клик )

Запускаем Orca Screen Reader
4) Включить автозапуск экранной клавиатуры.
"Onscreen keyboard" для вывода клавиатуры на экран
5) Установить дополнительное ПО для увеличения участков экрана.
Keyboard -> Universal Access -> Zoom in
